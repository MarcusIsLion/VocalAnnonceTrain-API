VocalAnnonceTrain API

Grâce à cette API, vous allez pouvoir réaliser des annonces de train personnalisées pour simuler une voie en gare ou dans un train. Pas de fichier audio à télécharger, tous se fait sans prendre de place sur votre machine !

Cette API à été développé par MarcusIsLion et aidé par romainZer

Départ d'un train

Dans cet exemple, vous allez découvrir comment utiliser l'API pour obtenir un message vocal pour le départ d'un train.

Pour commencer, le fonctionnement de l'API correspond à un site web. Pour obtenir une annonce vocale, vous devez effectuer une requête internet comme suit :

https://www.VocalAnnonceTrain.api/

Cette adresse vous redirige ici, car elle est considérée comme la page d'accueil de l'API.

Toutefois, vous pouvez appeler d'autres pages comme "TrainDeparture".

https://www.VocalAnnonceTrain.api/TrainDeparture

Attention tout de même, en appelant cette page, vous devez lui transmettre 8 paramètres :

  • Le type du train (TGV, TER, INTERCITE)(1 type max)
  • Le numéro du train (XXXX, numéro à 4 chiffre maximum)
  • Le type de voie (V [= Voie], Q [= Quai], P [= Plateforme])(1 type max)
  • L'indicateur de la plateforme ([A, B, C, ..., Z]/[1, 2, 3, ...,26])(1 indicateur max)
  • La station d'arrivée
  • L'heure de départ (00, 01, 02, ..., 23)
  • Les minutes de départ (00, 01, 02, ..., 59)
  • Le type du jingle (Station, IBTGV [ = In Bord TGV], IBTER [ = In Bord TER])(1 type max)

Voici donc un exemple de requête :

http://127.0.0.1:5000/TrainDeparture?train_name=TGV&train_number=234&plateform_type=Q&plateform_indicator=H&arrival_station=Marseille&departure_Hours=10&departure_Minutes=30&TypeJingle=Station

Passage d'un train

Dans cet exemple, vous allez découvrir comment utiliser l'API pour obtenir des informations sur le passage d'un train.

Pour cela, vous pouvez appeler la page "PassingTrain" en spécifiant les paramètres suivants :

  • Le type de voie (V [= Voie], Q [= Quai], P [= Plateforme])(1 type max)
  • L'indicateur de la voie ([A, B, C, ...,Z]/[1, 2, 3, ... 26)(1 indicateur max)

Voici un exemple de requête :

http://127.0.0.1:5000/PassingTrain?Sidding_type=voie_principale&Sidding_indicator=A

Suppression d'un train

Dans cet exemple, vous allez découvrir comment utiliser l'API pour supprimer un train.

Pour cela, vous pouvez appeler la page "DeleteTrain" en spécifiant les paramètres suivants :

  • Le type du train
  • Le numéro du train
  • La destination du train
  • L'heure de départ (heures et minutes)

Voici un exemple de requête :

http://127.0.0.1:5000/DeleteTrain?train_name=TGV&train_number=234&train_destination=Marseille&departure_Hours=10&departure_Minutes=30
